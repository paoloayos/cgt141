Terminology

Processor - GHz:

The electronic circuitry that executes directions comprising a pc program. It tells the computer what to do. The more powerful processor you have the faster it can complete its tasks.

Graphics Card:

An expansion card which produces a feed of output images to a show contrivance. It performs quick math calculations while freeing the CPU to compose other tasks.

Ram:

Random-access memory is computer memory that can be read and changed in any order, typically utilized to store working statistics and code.

Storage:

Used to retain digital data. Stores data for long-term use permanently. It stores all the files, applications and operating system.

Resolution- 1080p 4k:

The number of pixels in a digital image or display. So 1080p is 1080 pixels in the screen which would make the image or video more clear than 720 pixels. 4k is 4000 pixels.

Refresh rate- 60hz 240hz:

The frequency with which the image on a monitor or screen is refreshed. The more hertz the faster your pc will display images.

High dynamic range:

Allows the computer the display a broader spectrum of colors and contrasts.

100% Adobe RGB:

The display of colors. 100% will allow your computerto display most of the colors.

IPS:

Intrusion prevention system works to protect your computer from threats.

Ports:

Allows you to connect another device or other memory storage to your computer. Examples: sd card reader, thunderbolt, usb, audio, hdmi

Touch screen/ pen touch:

Allows the user to use their fingers or a stylist on the screen to move the cursor around instead of a mouse.

Biometrics:

Used to identify the user by their physical or behavioral characteristics to grant them access to the system. Windows hello, finger print login, face id
